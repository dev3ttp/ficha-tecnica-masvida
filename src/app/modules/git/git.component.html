Configuración inicial de Git

    - Acceder a la url https://git-scm.com/
    - Descargar el instalador git.
    - Ejecutar el instalador que se acaba de descargar dejando todas las configuraciones por defecto.
    - Abrir la consola de Windows y digitar el comando “git - -version” para verificar la instalación.
    - Generar ssh de acceso a los repositorios git con el comando “ssh-keygen”.

Es importante dejar el nombre del archivo y las contraseñas vacías, para ello solo debe

presionar la tecla enter en todas las opciones; en caso de existir una ssh se debe escribir

“y” en la opción “Overwrite” para generar una nueva ssh y reescribir la existente.

    - Acceder a la ruta “C:\Users\”nombre_usuario”\.ssh”, en caso de no ver el directorio .ssh
    debe habilitar carpetas ocultas.
    - Abrir con un editor de texto el archivo “id_rsa.pub”, el cual contiene la key de acceso,
    se debe copiar todo su contenido.
    - Luego debe acceder a la url https://gitlab.com/users/sign_in con las siguientes
    credenciales:
    usuario: felipe.rios@totalpack.cl
    clave: TpackW$1
    - Al acceder pincha el icono superior derecho y seleccionar la opción “Settings”.
    - Seleccionar la opción “SSH Keys” en el menú lateral izquierdo.
    - Pegar el contenido de nuestro archivo “id_rsa.pub” y pinchar “Add Key”.
    - Si nos muestra la siguiente ventana, se agregó correctamente la ssh, de lo contrario se
    deben repetir los pasos anteriorers de generación ssh.
