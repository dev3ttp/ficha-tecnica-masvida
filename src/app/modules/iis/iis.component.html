Configuración IIS (Internet Information Service)

    - Para iniciar debemos acceder a panel de control y buscar programas y características.
    - Pinchar la opción agregar y desactivar características de Windows.
    - Buscar Internet Information Services y verificar si el recuadro está pintado de color negro
    o tiene un check, de lo contrario se debe seleccionar el cuadrado y poner aceptar.
    - Al finalizar la instalación en caso de no tener habilitado, buscar en la lupa de Windows IIS
    y seleccionar “Administrador de Internet Information Services”
    - Al seleccionar Sitios, si se ve alguno como en la imagen, se recomienda detenerlo, para
    no tener conflictos con el puerto 80.
